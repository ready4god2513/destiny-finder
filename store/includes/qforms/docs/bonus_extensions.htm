Bonus Extension Libraries

The bonus extension members of the qForms API are split in to several files. These files are located in the "/qforms/" sub-directory. These extension libraries greatly enhance the normal functionality of qForms, giving you access to easy-to-use cookie methods and methods for handling bit-type operations.

Bits Extension Methods Library

  1. Methods
    • getBits([boolean useValue])
    • setBits(int value, [boolean useValue])

Cookies Extension Methods Library

  1. Functions
    • _createCookiePackage(object struct)
    • _deleteCookie(string name, [string path], [string domain])
    • _getCookie(string name)
    • _readCookiePackage(string package)
    • _setCookie(string name, string value, [string expires], [string path], [string domain], [string secure])
  2. Methods
    • loadFields()
    • saveFields()
    • saveOnSubmit()

WDDX Extension Methods Library

  1. Functions
    • __serializeStruct(object struct)
    • __wddxValue(string string)
  2. Methods
    • qFormAPI.serialize([string exclude])
    • obj.serialize([string exclude])
